# 前言 #

## 你，到底怎么了？ ##

人们总是对自己的生活设限。对于自己不足的地方，他们勇于承认，但却不敢面对内心真正的向往。明明心里想做得更多，但还是消极以对。而且，越来越多人定好计划后，很快又以各种形式退缩。总之，低成就是日程生活中“**惯性堕落**”的延伸，例如看不完一本书、老是拖延减肥计划、无法按时回信，等等。

在低标准的环境中，我们理所当然地降低对生活的期待和情感上的向往，降低对人际关系、自我认知、学识教养，以及个人成就的期待和需求，几乎无人不受影响。

无法突破个人障碍或社会氛围的人，渐渐变得麻木疏离，安于较低的品质，习惯与“有就好”的妥协。

改变是可能的，然而光是“想要”改变是不够的，你至少需要表明决心，并且承诺会持续努力一段时间。很不巧，会自我限制成就的人，通常会习惯性地避免这种承诺。为了发挥最大潜能，你一定要摒除积习，学习新的技巧。明确知道自己要什么，并依技术循序渐进，就一定会成功。否则，就像一直撞玻璃的苍蝇一样，永远都会遭遇无形的障碍。

不想发挥自己的潜力，只让它维持在最低限度，这就是一种退缩，也等于是放弃了自己的最大权益。相对来说，“成就自我”不只是人生的主要任务，它也可能是你一生中最惊险刺激的探索旅程。

#I 这是你要的自己吗？#

##1.为何我们习惯自我挫败##

> 人都一样。将目标设低，然后达成期望。
> —— Patrick White, Voss

省略步骤的处事策略，可能会导致自我设限的行为模式。如果你不愿在必要时耐心且有系统地做事，那么，你就不会努力打拼，宁愿坐享其成。**再伟大的事情都有它琐碎的部分，省略步骤绝对是不可行的。**

###你是不是这种人？###
+ 凡是力求简便，只做最低程度的努力，即使是很重要的事情
+ 花很多时间“准备”工作、或找别人“代替”你的工作，但就是不“投入”工作
+ 做事不持久，常常徒劳无功
+ 无法乐在其中，即使是面对你人生中重要的活动或人际关系
+ 犹豫不决，无法作决定
+ “订立”计划、“谈论”计划，但就是无法“贯彻”计划
+ 无法达成任何长期目标，因为缺乏适当的计划和持久度
+ 总是对新鲜事物感到兴奋莫名，当新奇感消失后便开始失望倦怠
+ 追求目标的动机错误，变动频繁。很快感到无聊，总是渴望开始新的事物
+ 无法完成重大计划，无论是不是自己喜欢的事情
+ 接近成功的时候，就萌生放弃的念头
+ 做事拖拖拉拉
+ 只参与低于你真正能力的工作，因为这样不必全力以赴
+ 常被非传统的事物吸引，一心回避传统、按部就班的工作计划或老板的命令
+ 有些事情你认为一下子就会过去，却老是深陷其中
+ 总是害怕无法达成自我或他人的期望
+ 总是担心自己是虚有其表，而且即将被拆穿
+ 无法力争自己内心真正向往的东西，因为你不想失望或失败
+ 对于“成功的先决条件”的认知不切实际
+ 避免作出重大承诺，因为想要有更多选择
+ 老是分不清“机会”和“冒险”的差异，习惯性承担无谓的风险，或是太过害怕失败，或是在两者之间摇摆不定
+ 把失败归咎于运气不好或是他人过失，不肯承认自己的错误
+ 觉得自己有社会适应不良症，认为自己落后于同辈，无法实现远大的理想
+ 总有一种“快没时间了，自己却还没开始”的感觉
+ 常陷入沮丧的情绪中

###逃避，真的可以换来永远的自在吗？

在追求理想的过程中，难免会遇到必须下赌注或作出重大承诺的时候。

###放弃机会，你敢保证将来不后悔？

有时候我们很难分辨什么是自己真正想要的，什么是不想要的，因为这牵涉太多外在的期望和压力。然而，你必须在心底有条明确的界限，一旦厘清了，你就会像其他人一样渴求改变，朝着目标挺进。

###故意选择“低成就”的人

因为自我设限，导致低成就的高潜质群族（Self-limiting High Potential Persons, 简称SLHPPs）。

在各方面都限制自己发挥的程度，这种行为就叫做“自我设限”。没做好“自己可以做的好”的事，就是一种自我设限的行为。

自我设限的行为是一种“选择”。

首先，你不一定知道或意识到“选择”的存在，你可能已经做了选择却不自知，甚至当你决定不去选择时，也是一种选择。

不论是不是天性使然，限制你自己、不做完全努力，就是一种自暴自弃的行为。

有些自我设限的后果立刻显现，有些则要经过一段时间才看得出来。以下是人们日常生活中自我设限的模式：

+ 他们隐约感觉到，有些东西好像不见了，他们感到空虚和失落，但并不深究原因、找到解决方法。有些人就是不想深入探索自己的好奇心、兴趣、雄心或天职。因此他们的生活失去力道与重心，没有使命，也没有热情。

+ 有些人觉察到自己的愿望，但将它束之高阁，变成一种“假设的可能”，仿佛拥有梦想就足够，真要付诸行动，简直是不可能的人物。

+ 有些人跨出了一些，他们研究过追求梦想的可能性，但很快便放弃了。他们没有坚持下去，将抛弃梦想视为“长大成熟”和“落入实际”的象征，或是以年纪太大或太轻、已婚或未婚、胆量不够为理由，放弃追求梦想。

+ 有些人非常确定，也许现在还做不到，但等到“环境允许”，他们就会行动。但环境的压迫一波接着一波，因为种种原因，永远都不是“最好的时机”。他们从来没有放弃，只是从未“开始”。

###就这样，你困住了自己

自我设限的高潜质族群总是一再重复自我挫败的做事方式。如果你习惯性地在某个重要领域中限制自己的成就，那么，在其他领域你也可能会裹足不前。这样的倾向，将会导致全面性的自我设限行为。然而，自我设限没有固定的模式，你可能在自己或别人身上发现多种符合的特性。一下叙述未必罗列出所有的可能，只是让你了解，在日常生活中，人们会习惯性避免或错过的机会。

+ 沉睡者（sleepers）
+ 漂流者（floaters/coasters drifters）
+ 进退两难者（checkmates）
+ 绝不冒险者（extreme non-risk-takers）
+ 拖延者（delayers）
+ 功亏一篑者（stop-shorts）
+ 自我怀疑/自我打击者（self-doubters/self-attackers）
+ 表面功夫者（charmers）
+ 超级冒险者（extreme risk-takers）
+ 反抗者（rebels）
+ 怀才不遇者（misunderstood geniuses）
+ 宁缺勿滥者（best-or-nothings）

你尽力融入人群，却忽视内心被压抑的东西。

##2.孕育平庸的摇篮##

> 也说不出怎么变成这样的，就是在睡意迷蒙中远离了正确的方向。
> —— Dante Alighieri, The Divine Comedy

###别被标签害了你

如果只是一味退缩逃避、带着模糊的期望或需求，被形容为“有才华”，将会产生复杂且不可预料的结果。只有当人们清楚地期待某种成就的情况下，“才华”才显得有意义。累积的成就无形中增加了有才华者的自信，相对的也让他们更愿意接受挑战。

如果没有明确的要求和支持，被贴上“天才”或“聪明”标签的人，就如同贵族后裔一样——空有遗传的显赫身世，和实际作为无关。被贴上优秀的标签，通常会伴随着无形的期望，然而当缺乏明确的要求和支持时，这些标签变得摇摇欲坠。

一方面这些标签告诉你，光是被称赞聪明和有才华就够了；另一方面，被赞美得越久，你就越紧张于将被要求有所作为。如果时机真的来了，我却达不到他们的期望，怎么办？要是机会来了，却没有想象中简单，怎么办？要是别人跟我一样好，甚至比我优秀，那该怎么办？如果我根本没有这些才华，怎么办？

在处境艰难的时候还能咬紧牙关、坚持到底的人才能了解，琐碎重复、辛苦乏味，其实都有它们的价值。靠坚持而获得成功的人，最后通常都会取代光有天赋的对手。

###故意“不尽力”的人

社会心理学家发现：针对表现不佳，“没有努力”居然是普遍被拿来使用的合理解释之一。

以“没有努力”进行合理化的内在诱因，会产生一系列的心理变化：***我失败了，但因为我没尽力，所以这次不算。如果不去尝试，当然就不会失败。不去尝试，我还是可以维持“要是我尽力，就会成功”的形象。如果我尽力却失败了，我就再也不能以此为借口，所以，最好还是不要尽力。***

***他们避免失败的两种策略，就是自我保护的典型。***每一种策略都让他们无法得到成就的满足感，在这种心态下，挑战最高或最低极限其实结果都一样。

###编织借口，成了一种新兴习惯

“保持完美形象”是自我设障者最重要的目的。想出无数个理由和借口，解释自己的表现不佳。

不论这些借口是否被全盘接受，“编织借口”变成一种新兴的习惯。然而，“编理由”就是一种自我挫败和自我设限的行为。怪罪他人，等于贬低自己的掌控力，把你个人的力量隐藏于无形。

成就是不断努力累积的结果，虽然中间偶尔会有灵光闪现的神奇时刻，但整体来说，就是毅力的累积。无论踏入任何阶段，无疑都需要持之以恒。

人生也是一样的道理。我们从熟知的地方进入未知的领域，如果你抗拒这样的过程，就是阻断了自己的进化。

##3.你的魔法怎么不见了##

> 如果把一个人的生活想象成一个房间，很显然，大部分的人都只熟知房间的某个角落，就是他们来回走动的那块地板。
> —— Rainer Rilke, Letters to A Young Poet

魔法，也有它的限制。光靠天赋或微薄的努力，终究会有你竭尽全力，也无法撑下去的时刻。

突然间事情不再那么简单，在某事某地，某些人就是比你聪明、比你知道得更多、比你更努力求上进。无论是在学校或事业发展的初期，你都必须决定要怎么走下去。***如果想维持在最佳状态，你必须找到充实自己的新方法。***

###魔法不再的时刻

当云雾散去，一切变得明朗，两个基本选择也浮现眼前：坚持到底，还是尽量闪避拖延、寻找最柔软的地方降落？

###关于工作的12个愿望

不要因为害怕、恐惧，而拒绝某些工作，你会因此丧失成长的机会。

###你该追寻什么样的工作？

你要追寻的是什么样的工作？这个问题的答案，就在与“什么对你是最重要的”。

###总是彩排中的人生

当你年轻的时候，未来是向前无限延伸的，明天似乎永远不会来。渴望美好前景却遍寻不着的状况下，这些拖延者开始安于他们一开始视为过渡时期的状态。这些早期的决定会限制他们日后的选择，他们也会把这种惯性拖延带进职场。拖延者通常只为了钱工作，或为了必须工作而工作，所以他们会尽量避开要求太多的工作。

###什么？朝九晚五？溜吧

工作无法满足自我成长的需求，其实不见得是个大问题。事业并非自我实现的唯一路径。

###关键，就在细节

看到这里，我们应该很清楚，造成自我设限和低成就的原因就是害怕失败。因为害怕失败，所以不去冒险；但是当你开始逃避，你就已经限制了自己的人生。

***逃避比失败本身更危险。***

这次没完成的事，下次就更难完成。必然的结果是：1.如果你逃避去做某件事，下次很可能会再度逃避；2.当你逃避某个人生的重大课题，就更难去面对下一个决定。

电影制作人、舞蹈家、剧作家、木匠、雕刻家、运动员，甚至是修车工人，他们的表现之所以出色，就是因为注重细节。他们不断从失败中吸取教训，持续地学习。

###老是累得跟狗一样

SLHPPs都是好奇心强的享乐主义者，向往着休闲旅游，但最后往往只变成窝在家当懒骨头而已，真正的享乐和自我实现就是与他们无缘。要享受休闲时光，首先，你得先把时间空出来，而且绝不能妥协。这表示你必须在之前把手边的事情完成，才不会分心。要及时完成任务，你需要完美的时程规划，而且要妥善利用时间。另外，事前要先考虑清楚：什么才是你真正想做的？需要花多少钱？什么时候可以成行？要怎么做？做这些决定其实不用花太多时间，但是SLHPPs凡事不喜欢动脑筋，就连这点时间也做不到，所以最后通常还是妥协，日复一日地过着相同的生活。

渴望的放松和乐趣，是需要妥善规划的，不然很容易毫无目标，变成只是在熬时间。检视一下自己放假时都做些什么吧。也许一直以来都是老样子，根本就没有更改过。

如果你觉得自己没有资格放假享乐，因为有一对事情还没做完，那你就错了。***创造快乐的人生就跟完成工作目标一样，需要技巧，所以寻找能让自己身心满足的休闲，也是一门学问。***更何况，每个人都需要生活乐趣。如果没有充电和放松的机会，怎么会有力气应付生活上复杂难缠的挑战呢？好好计划这个星期天的假期，而且，无论任何都要好好实行。然后，把规划假期的这份专注同样用在工作上，努力实行。致力于这样的练习，可以让你的人生更加美好。

###小心“后悔”这种感觉

要小心“后悔”这种感觉，它是个狡猾的家伙。从中汲取教训就好，但千万不要沉溺其中，一不小心，它可能会诱使你为了抢救失去的时光，做出一连串错误的尝试。你会开始以最奇怪的角度，从最远的地方丢掷小沙袋。你也可能在自认无法弥补错误之后，彻底被击垮，从此动弹不得。很多“自我怀疑/自我打击者”，通常都会在这个地方倒地不起。

虽然大家都希望在错误造成前可以先学习避免，但事实上这是不可能的。如果你对自己太过严厉，只会让过去的错误成为今日和未来的阻碍。你必须对后悔施以反击，否则它就会毁了你。

很多SLHPPs孤立自己，让自己忍受“不够好”的状态。如果你的生命出现了这些危机，你应该感到庆幸，并且善加利用，成为日后的转机。不要等到临死前才惊觉：“天啊，我的一生就这样过了。”你是自己生命的守门人。每一天，你都可以自己决定要怎么活。


#II 你变了！#

##4.谁说本性难移##

> 我们必须坚持与以下两种痴狂对抗：相信自己无所不能；相信自己一事无成。
> Alain, Alain on Happiness

***没有人可以轻轻松松就改变习惯，尤其是对付那种难缠的积习。如果你永远只想不费吹灰之力就完成任务，那么你很快就会失望。那些阻挠你看清自己潜力的鸵鸟心态，也会阻挠你改变。

真的这么难吗？当然不是。你只是缺乏某些技巧和认知。

###改变的秘密真理

+ 选定目标集中努力，改变会比较容易
+ 想要改变，你必须抱着长期抗战的准备
+ 当你付出了努力却没有明显效果的时候，坚持下去，不要觉得麻烦、困扰或想放弃
+ 混乱和挫折，证明你正在改变
+ 你无法改变过去
+ 问题从来就不在于“你是谁”，而是：你选择要做什么。自己有能力选择新的方式，并且付诸行动。
+ 你需要全神贯注地思考、准备、决定你的行动计划，并且持之以恒
+ 改变不是“发生在你身上的事”，而是“你要去做的事”
+ 改变是一种学习过程：学习以不同的方式去做事
+ 在学习过程中，失败是必要的
+ 改变没有规则可言，出现的速度也不会永远都一样
+ 改变的过程你会一再受挫
+ 企图同时尝试太多改变，只会适得其反
+ 在掌控范围内慢慢改变
+ 详细规划，改变才能顺利进行
+ 改变就建立在改变之上的

###三个管家的故事

***不去尝试就是一种失败。***成功的唯一方法就是不断尝试。才华本身并没有太大的意义，再好的手段和方法如果不表达出来，也无法成就任何事。

事实上，完美的人生建筑在一次又一次的失败上，没什么好悲叹和闪躲的。想要体验真实人生，就要欣然面对现实。唯有接受挑战、承担风险、经历失败，你才能学会生存之道。如果你总是不费吹灰之力就完成事情，便不会有足够的雄心，也会永远将自己限制在相同的领域内。

###珍惜改变的代价

开始一个新阶段是很不同意的，尽管你已经做了准备，尽管它是你的梦想。然而，如何看待变动带来的不愉快，你的态度将决定你能否成功跨越障碍。

为了追求梦想，我慢慢吸收变动带来的震撼和不适应。这些负担是很累人、很不愉快的。我是否觉得自己无法胜任？是的。但一切是否值得？是的。付出一些必要的代价之后，我感受到熟练和胜利的成就感。

###你正在哪个改变的阶段

心理学家研究出人类改变过程的五个阶段性模式。

1.***思考前期***

这是你开始感到不安的阶段——这也是改变的契机。在这个时期，你不一定真的想要改变，也没有任何成形了的具体计划。

你觉得有些事情不对劲，但不一定知道错在哪里。针对自己的困扰，你尚未理出头绪，也没有找出解决方法，更别说是实际行动。反之，你只是觉得自己正经历低潮，或是必经的不安和挫折时期。你觉得怪怪的，感觉有东西在扯你的后腿，但你的认知也仅止于此。

在这个阶段，“改变”还是个模糊、遥远、不太可能的抽象概念。从你抱怨某事很麻烦、希望情况可以快点改变的方式，可以看出端倪。

2.***思考期***

到了这个时期，你已经了解到自己出了问题，而且开始思考问题所在，自己有没有办法解决等等。此时你还是不想改变，但开始在拒绝改变与有所行动间斟酌。你常常为了要不要采取行动而挣扎不已，怀疑自己是否有改变的能力，因而心情忧郁不定。你还没作出改变的承诺。本时期最重要的两个特质是不确定和犹豫不决。

3.***准备期***

当改变成为势在必行的事实，你开始采取一些必要的步骤，这时，你就正式迈进准备时期了。即使你的行动是暂时的，或是你仍然习惯有所保留，无论如何，这是一个积极活跃的阶段。你开始搜集资料、打电话询问课程、注册补习，想要重回学习时光。你重新检视自己的行程，腾出时间，开始以改变为前提来思考和行动。

准备期的重要作用，是让自己的内心慢慢适应改变后的生活。你需要想象一下，改变之后，自己的人生会变成什么样子，在心里预演未来的生活，这会让你更坚定信念，接受更高的要求和期望。也就是说，你终于能够坦然面对自己的恐惧。

4.***行动期***

在本阶段，改变已经完全融入你的生活。多年的挣扎和酝酿，刹那间豁然开朗。在改变的过程当中，你会和支持你的人建立深厚且相互依存的情感。

5.***维持期***

历经上一阶段改变的混乱后，接着到来的也是一个非常活跃的时期。维持，虽然听起来有点单调，却是改变过程中最重要的部分。因为此时你将尽所有努力，来巩固这个新的改变。这是个一切趋于稳定的阶段，改变已经深入影响你的生活。你花越多的时间来维护和坚定你的新习性，一切就能越快决定。没多久你就可以到达一个怡然舒适的状态，用全新的习性运作你的人生。

在每一个阶段，你都要试着说服自己，也说服更多人支持你的计划，并且相信自己选择了正确的道路，试着从心底开始改变。

你无法为了其他人或是一个不相干的原因而改变自己，因为你不是从心底彻悟，向往改变。改变关乎你的心。只期望凭借外力，甚至神力的帮助，都是没有用的。前方有无数错误的指引和绝境，你必须持续努力，和时间拔河，不能浪费太多时间。因此，如果你集中心力在正确的事情上，整个过程会容易许多。


##5.改变，就是这么简单##

> 从这里之后，就不能回头了。你必须到达目的地。—— Franz Kafka, The Great Wall of China

***当你些微察觉到有什么不对劲的时候，你已经进入了改变的最初阶段。***

此时，通常你已经努力好一阵子，也开始了解到，你埋没了自己的潜力。如此一来，你不只经历了思考前期，更迈进了改变的第二阶段——思考期。这并不容易，因为这两个阶段是改变中历时最长的过程。现在，你即将进入第三阶段——准备期，根据自己的目标，订立明确计划。为了成功改变，你必须坚守完成这个阶段，采取必要的行动，并持之以恒地稳固你的成果。
